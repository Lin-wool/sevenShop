# 购物车、订单及管理员功能修复

## 为什么

当前存在多个功能缺陷：购物车数量输入失焦后数据未正确保存；商品购买弹窗不支持数量输入；管理员订单列表缺少下单人信息无法有效处理订单；管理中心统计数据未正常实现且需要支持跳转和用户管理功能。

## 变更内容

1. **修复购物车数量输入问题** - 修复失焦后数量未正确保存和小计未重新计算的问题
2. **商品购买弹窗添加数量输入** - 在商品规格选择弹窗中添加可输入的数量字段
3. **管理员订单列表添加下单人信息** - 显示下单人昵称以便识别订单来源
4. **修复管理中心统计功能** - 修复统计数据获取逻辑，支持点击跳转到对应列表
5. **新增用户管理页面** - 添加用户列表和禁用/启用功能

## 功能 (Capabilities)

### 新增功能

- `user-management`: 用户管理功能，支持查看用户列表、禁用/启用用户

### 修改功能

- `cart-quantity`: 购物车数量输入逻辑修复
- `product-purchase`: 商品购买弹窗增加数量输入功能
- `admin-order-list`: 订单列表添加下单人信息
- `admin-stats`: 管理中心统计数据修复和跳转功能

## 影响

- **前端**: MobileCart.vue, Mobile.vue, AdminOrders.vue, Admin.vue, MobileAdmin.vue
- **后端**: 可能需要添加用户列表API
- **用户体验**: 修复多个交互问题，提升管理效率
