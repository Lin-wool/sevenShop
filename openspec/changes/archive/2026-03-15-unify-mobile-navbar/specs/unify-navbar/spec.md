## 新增需求

### 需求:统一移动端导航栏
系统必须使用统一的 MobileTabbar 组件作为所有移动端页面的底部导航栏。

#### 场景:商品页使用统一导航栏
- **当** 用户访问商品页
- **那么** 页面底部显示统一的导航栏，包含商城、订单、购物车、配送、我的

#### 场景:订单页使用统一导航栏
- **当** 用户访问订单页
- **那么** 页面底部显示统一的导航栏，当前"订单"tab高亮

#### 场景:我的页使用统一导航栏
- **当** 用户访问我的页面
- **那么** 页面底部显示统一的导航栏，当前"我的"tab高亮

#### 场景:地址页使用统一导航栏
- **当** 用户访问地址管理页
- **那么** 页面底部显示统一的导航栏，当前"配送"tab高亮

#### 场景:购物车数量实时更新
- **当** 用户添加商品到购物车
- **那么** 导航栏购物车图标上显示正确的数量badge
